MAG SDK 使用文档

MCP Agent Graph - 基于图的智能体开发框架 Python SDK

项目概述

MAG (MCP Agent Graph) 是一个基于图的智能体开发框架，让您能够通过可视化的方式快速构建和部署复杂的AI智能体系统。

图形化设计

通过直观的图形界面设计智能体工作流，每个节点代表一个具有特定功能的智能体。

MCP集成

无缝集成MCP服务器，为智能体提供丰富的工具和能力扩展。

图嵌套

支持将整个图作为节点使用，实现层次化的智能体架构设计。

导出为MCP

将图导出为标准MCP服务器脚本，实现智能体的独立部署。

快速开始

安装

1

安装MAG SDK

使用pip从PyPI安装最新版本的MAG SDK

pip install mcp-agent-graph
2

导入并启动服务

在Python代码中导入mag模块并启动后台服务

import mag

# 启动MAG服务
mag.start()

# 检查服务状态
if mag.is_running():
    print("MAG服务已成功启动！")
3

开始使用

现在您可以开始使用MAG SDK的各种功能了

# 查看可用的图
graphs = mag.list_graphs()
print(f"可用的图: {graphs}")

# 查看可用的模型
models = mag.list_models()
print(f"可用的模型: {models}")

主要功能

MAG SDK 提供了以下主要功能模块：

  • 系统管理 - 服务启动、状态检查、关闭服务
  • 图管理 - 图的创建、编辑、运行、导入导出等完整生命周期管理
  • 模型管理 - AI模型的配置、更新、删除等管理功能
  • MCP管理 - MCP服务器配置、连接、工具管理
  • 会话管理 - 智能体执行会话的查看、管理、删除
